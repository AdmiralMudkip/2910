Ideas for functions:

function checkLocation  - checks boat location, if it's an empty tile, bad tile, or win tile

function win - called if checkLocation reports the boat is on a win tile

function loss - called if checkLocation reports the boat is on a loss tile

function moveLeft - called if left arrow is pressed

function moveRight - called if right arrow is pressed

function moveUp - called if up arrow is pressed

function moveDown - called if down arrow is pressed

\\\\\\\\\\\\\ - UI STUFF


function toMainMenu - applied to any button that takes you to the main menu

function toLevelSelect - applied to level select button on the main menu

function toHighScores - applied to pause menu highscores button as well as main menu highscores button
